## Usage

1. Clone the repository and run the `install.sh` file.

``` bash
$ git clone https://github.com/flyingluscas/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./install.sh
```

2. Reboot your system after the installation is finished!

``` bash
$ sudo reboot
```
